# DATA-SCIENCE


This repository contains data  projects built using Python and Jupyter Notebook.

Tools Used
Python
Pandas
Matplotlib
Seaborn
Jupyter Notebook

Projects

Pizza Sales Analysis
Analyzed a pizza store dataset containing 48k transactions to understand sales patterns, peak order hours, and product performance.

Key Insights
• Large pizzas were ordered most frequently
• Classic category generated highest sales
• Peak orders occurred during lunch and evening hours

Customer Churn Analysis
Analyzed telecom customer data to identify factors influencing customer churn.

Key Insights
• Month-to-month contracts had highest churn
• Higher monthly charges increased churn probability
• Customers with longer tenure were less likely to leave
